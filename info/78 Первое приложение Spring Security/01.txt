78 Первое приложение Spring Security

После распаковки приложения оно уже работает
Создаем контроллер и шаблон (представление)
Запускаем приложение с помощью
src/main/java/project/SpringSecurityApplication.java

Примечание
(без подключения БД и настройки application.properties
приложение не запустилось)

В браузере пишем запрос
http://localhost:8080/hello
и нас, по умолчанию, автоматически перенаправляет на
http://localhost:8080/login
блокируя доступ ко всем адресам

Во время запуска мы видим в консоли, что Security уже создал
пользователя по умолчанию
По умолчанию логин: user
Using generated security password: 6af89fd7-481f-42b4-97e9-a203b6452a23
