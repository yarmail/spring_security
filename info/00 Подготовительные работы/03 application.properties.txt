application.properties


В этом файле мы должны описать
- конфигурацию нашего источника данных - jdbc драйвера
который мы подключали в pom

- конфигурацию Hibernate
- добавляем диалект Postgres
- включаем отображение запросов hibernate к базе
в консоли
- последняя настройка связана с сессией

- могут быть и другие настройки

Примечание
Чтобы файл настроек адекватно относился к русскому языку
я изменяю кодировку файлов .properties
Для этого я изменил следующую настройку:
File - Settings - Editor - File Encodings
Default encoding for properties files:
Было: Properties Default: ISO-8859-1
Стало: UTF-8
