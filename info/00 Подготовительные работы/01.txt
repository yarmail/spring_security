01 Подготовительные работы

Используем Spring initializer
https://start.spring.io/

(см. Spring Initializr.png)

Spring Boot может собираться в jar файл и использовать 
встроенный  веб-сервер. 
Если мы будем использовать нашу программу
на чужом сервере мы можем выбрать war

----

Добавляем стартеры (add dependencies) :

Spring Web - основной стартер
(должен содержать Spring Core, Spring web, Spring MVC)

Thymeleaf - для шаблонов, представлений

Spring Security - аутентификации и авторизации

Spring Data JPA - для работы с базой данных

PostgreSQL Driver - для работы с базой данных

Validation - для проверки данных


---

Далее мы делаем Generate и скачиваем архив проекта,
который будем разархивировать и класть в папку проектов

Далее заходим в Idea, открываем папку и подтверждаем Maven,
что это Maven проект, ждем пока мавен все загрузит

Копируем, проверяем или создаем .gitignore и README.md
в корне проекта
